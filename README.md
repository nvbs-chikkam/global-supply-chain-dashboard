# Global Supply Chain Command Center

## Architecture Overview
An end-to-end Power BI logistics dashboard engineered to monitor global revenue distribution, transportation modes, and defect rates across major enterprise hubs.

## Technical Stack
* **BI Tool:** Power BI
* **UI/UX:** Custom Dark-Mode Interface, Native Visual Borders
* **Data Processing:** DAX, Power Query

## Core Features
* Real-time transit mode efficiency tracking.
* Hub-level defect rate analysis.
* Year-over-year transaction performance metrics.
